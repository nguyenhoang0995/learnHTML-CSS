1. HTML là gì?


HTML là chữ viết tắt Hypertext Markup Languageanguage, là ngôn ngữ được sử dụng rộng rãi nhất để viết các trang Web.
Hypertext là cách mà các trang Web (các tài liệu HTML) được kết nối với nhau.
Và như thế, đường link có trên trang Web được gọi là Hypertext.
̛ tên gọi đã gợi ý, HTML là ngôn ngữ đánh dấu bằng thẻ (Markup Language), nghĩa là sử dụng HTML để đánh dấu một tài liệu text bằng các thẻ (tag) để nói cho trình duyệt Web cách để cấu trúc nó để hiển thị ra màn hình. Vậy HTML là ngôn ngữ sử dụng các thẻ để xây dựng nên khung sườn của 1 website.
